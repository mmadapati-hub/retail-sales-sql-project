## Objective
Analyze retail sales data using SQL and generate business insights.

## Tools Used
- SQL
- GitHub

## Project Structure
- schema.sql → database structure
- analysis_queries.sql → business queries
- insights.md → findings
- data.sql 

## Key Insights
- The analysis showed the overall revenue generated from all sales by combining product prices with the quantities sold.
- A few customers contributed significantly more revenue than others, making them the top paying customers.
- Some products consistently had higher sales volumes, indicating they are the most popular among customers.
- Monthly sales trends revealed how revenue changed over time, making it easier to identify periods of higher and lower sales.
- Comparing revenue across product categories helped identify which categories performed best and contributed the most to total sales.
- Overall, this project demonstrates how SQL can be used to transform raw sales data into meaningful business insights for better decision making.
